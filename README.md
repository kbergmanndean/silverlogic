# README
## Wall Project for SilverLogic

##General Info
User can view wall of posts by other users, then can sign up or log in to create posts.

##Technologies
This app was made using ruby 2.7.3 and react 17.0.2

##Setup
To run backend for this project:
```
$ rails s 
```
To run frontend:
```
$ cd ./client
$ npm install
$ npm start.
```
To run tests:
````
$ bin/rails test
```
